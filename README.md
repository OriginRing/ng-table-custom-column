# 自定义表格列

## 实现环境

- angular 14
- angular/cdk 14
- ng-zorro-antd 14

## 运行

```shell
git clone https://github.com/OriginRing/ng-table-custom-column.git
cd ng-table-custom-column
npm install
npm run start
```
